2. Apache Tomcat 설치

  • tomcat 9.0 설치
  • bin 폴더에 있는 startup.bat으로 실행, 'cd /'로 탐색기 돌아다님
  • conf 폴더에 있는 server.xml을 통해 포트번호 수정
  • webaspps 폴더에 html파일을 두면 서비스할 수 있음
  • 실행 시 JAVA_HOME or JRE_HOME 이라는 이름의 환경변수 필요함
